What is grid?

What is grid? IMAGE and keywords

A grid is a set of intersecting horizontal and vertical lines defining columns and rows. Elements can be placed onto the grid within these column and row lines. Elements can be placed onto the grid within these column and row lines. CSS Grid Layout introduces a two-dimensional grid system to CSS. Grids can be used to lay out major page areas or small user interface elements. Grid layout offers a grid-based layout system, with rows and columns, making it easier to design web-pages without having to use floats and positioning.
